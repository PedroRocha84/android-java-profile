# 🧮 Profile App

This is a simple **Profile Demo** app. This app is build with two activities where we can input our name and a short text as our Biography. Then both text will display in a new activity this allowed me to learn Android Lifecycle and savedInstances.

## 🚀 Features

- Simple UI with two activities
- Built using Java
- Log events and check using LogCat the inner activity and debugging.

## 🛠️ Built With

- **Android Studio**
- **Java**
- **XML Layouts**

## 🧠 Learning Goals

- Navigation between two Activities
- Work with buttons, `TextView`, `EditText`, `ImageViews` and `onClickListeners`
- Using resources and string values manipulations
- Learn how to update UI elements programmatically
- State Preservation and the usage of onSaveInstanceState() to save important data before the activity is destroyed
- Use onCreate() or onRestoreInstanceState() for restoring saved UI state
